Rendering to texture and virtual scenes.

Polycode allows Cameras to render directly to a texture and the creation of virtual scenes, which only render to texture.

Texture rendering overview.

Sometimes you want to render to a texture that you can use to apply to a mesh in a scene. You can use this to create things like mirrors, reflection maps, screens to other parts of a scene or a different scene entirely or use it for rendering 3d elements into 2d.

Creating render textures.

The class that we use for rendering to texture is the SceneRenderTexture class.

Creating virtual scenes.

Sometimes you want to ONLY render a scene to a texture without rendering it to screen directly. For example, you might want to render a 3d mesh onto a 2d element. In a Polycode, this is called a virtual scene.
