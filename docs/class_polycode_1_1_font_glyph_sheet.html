Detailed Description

Wraps a sheet of rendered font glyphs on a Texture.

Use in combination with TextMesh to render text from minimal texture creation.

Member Enumeration Documentation

enum Polycode::FontGlyphSheet::FontTextureGlyphMode
Enumerator
ANTIALIAS 

Regular anti-aliased font rendering.

Colour is pure-white for clean custom tinting with an alpha channel.

ALPHA_TEST 

Using distance-from-edge calculation as described in the Valve paper.

"Improved Alpha-Tested MagniÔ¨Åcation for Vector Textures and Special Effects" http://www.valvesoftware.com/publications/2007/SIGGRAPH2007_AlphaTestedMagnification.pdf

To make the most of this: set renderer->alphaTestValue = 0.5 set sceneMesh->alphaTest = true set sceneMesh->blendingMode = Renderer::BLEND_MODE_NONE;

Or use a custom shader - alpha values of 0.5 indicate the boundary.
